# client/assets

Runtime-loaded art. The `.gdignore` keeps Godot's importer out; everything here is
loaded with `Image.load_from_file` (see `client/scripts/asset_library.gd`).

- `ships/` — generated by `tools/artspike/composer.py` (M3.5 PR 1). Regenerate:
  `python tools/artspike/composer.py`.
- `stations/` — generated by `tools/artspike/stations.py` (M3.5 PR 2). Regenerate:
  `python tools/artspike/stations.py`.
- `bodies/`, `background/` — Distant Horizon Classic sprites (commissioned for
  Classic; we own them). Copied from
  `DistantHorizonClassic/client/sprites/{planets,background}`.
