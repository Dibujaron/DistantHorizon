# M3.5 PR 2 — Space Layer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the space layer's placeholder vectors with the real thing: lit ship
sprites from the PR 1 pipeline, a station exterior assembled from station data, Classic
planet/star sprites, restrained three-layer parallax, per-star DirectionalLight2D suns,
and throttle-scaled thruster plumes.

**Architecture:** `world_view.gd` keeps its immediate-mode `_draw()` for vectors only
(starfield tiles, orbit paths, dock rings, labels) and gains managed child sprites for
everything textured: bodies, stations (+ a decorative parked Longhorn), ships (+ plume
children), and one `DirectionalLight2D` per star whose rotation tracks the star→focus
direction each frame. A new `AssetLibrary` (plain RefCounted, loaded once by WorldView)
loads pipeline exports and Classic sprites at runtime (`Image.load_from_file`, no import
pass — `client/assets/.gdignore` keeps the editor out). Station exteriors are generated
by a new `tools/artspike/stations.py` through the PR 1 composer with authored heights
and berth anchors in meta.

**Tech Stack:** Python composer (PR 1), Godot 4.7 gl_compatibility, GDScript. Verification
via the harness automation socket (screenshot + dump) against a live server.

## Global Constraints

- Branch `m3.5-vibe-pass`, stacked on PR 1. Never commit to main.
- Hand-drawn assets (planets, star, starfield) skip normals and MUST NOT be lit by the
  suns: suns use `range_item_cull_mask = 1`; WorldView itself and every non-ship sprite
  set `light_mask = 2`. Only ship sprites carry `light_mask = 1`.
- World is y-up, screen y-down: screen pos negates rel.y (world_view.gd:74); ship sprite
  rotation = `-heading + PI/2` (textures point up; heading 0 = world +x).
- Quantize shader thresholds/steps stay identical to composer.py (client copy of the
  shader must not drift; it is copied from `tools/artspike/godot/lit_ship.gdshader`).
- Sprites use `TEXTURE_FILTER_NEAREST`; labels stay on the default filter (they live in
  `_draw`, not on sprite children).
- Preserve the manual projection + `_set_view_zoom` transition scheme — no Camera2D in
  this PR. Everything textured lives under WorldView so the crossfade carries it.
- Classic sprites are paid-for assets — copy them in, note provenance in a README.
- Scale constants (initial, eyeball-tunable): ships 1.33 world units per texture px with
  a minimum on-screen scale clamp of 0.5; station sprite spans ~1.6 × dock_radius.
- Don't block on user review: publish screenshots to the art-spike artifact (round 6),
  queue questions.

- [ ] **Step 1: Write `asset_library.gd`**

```gdscript
class_name AssetLibrary
extends RefCounted
## Runtime loader for client/assets (the tree is .gdignore'd - no import pass;
## same Image.load_from_file recipe the artspike toy scene proved). One
## instance is built by WorldView._ready; a SpriteSet bundles the lit
## CanvasTexture, the per-type ShaderMaterial (livery masks + bases), and
## meta.json (anchors, px size).

const SHADER_PATH := "res://shaders/lit_sprite.gdshader"


class SpriteSet:
	var texture: CanvasTexture
	var material: ShaderMaterial
	var meta: Dictionary

	func px_size() -> Vector2i:
		return Vector2i(int(meta.get("px_w", 0)), int(meta.get("px_h", 0)))

	func anchors(kind: String) -> Array[Vector2]:
		var out: Array[Vector2] = []
		for a: Variant in meta.get("anchors", []):
			if a is Dictionary and str(a.get("kind", "")) == kind:
				out.append(Vector2(float(a["x_px"]), float(a["y_px"])))
		return out


var _ships: Dictionary = {}
var _stations: Dictionary = {}
var _bodies: Dictionary = {}
var _star_layers: Dictionary = {}
var _shader: Shader = null


static func load_all() -> AssetLibrary:
	var lib := AssetLibrary.new()
	lib._shader = Shader.new()
	lib._shader.code = FileAccess.get_file_as_string(SHADER_PATH)
	var root := ProjectSettings.globalize_path("res://assets")
	for kind in ["mockingbird", "mockingbird_stock", "longhorn"]:
		lib._ships[kind] = lib._load_set(root + "/ships/" + kind)
	for archetype in ["ring_3berth_crane", "ring_1berth"]:
		lib._stations[archetype] = lib._load_set(root + "/stations/" + archetype)
	for f in DirAccess.get_files_at(root + "/bodies"):
		if f.ends_with(".png"):
			lib._bodies[f.trim_suffix(".png")] = _load_tex(root + "/bodies/" + f)
	for layer in ["small", "medium", "large"]:
		lib._star_layers[layer] = _load_tex(root + "/background/stars_" + layer + ".png")
	return lib


static func _load_tex(path: String) -> ImageTexture:
	var img := Image.load_from_file(path)
	if img == null:
		push_error("AssetLibrary: missing " + path)
		return null
	return ImageTexture.create_from_image(img)


func _load_set(dir: String) -> SpriteSet:
	var albedo := AssetLibrary._load_tex(dir + "/albedo.png")
	if albedo == null:
		return null
	var s := SpriteSet.new()
	s.texture = CanvasTexture.new()
	s.texture.diffuse_texture = albedo
	s.texture.normal_texture = AssetLibrary._load_tex(dir + "/normal.png")
	s.meta = JSON.parse_string(FileAccess.get_file_as_string(dir + "/meta.json"))
	if s.meta == null:
		s.meta = {}
	s.material = ShaderMaterial.new()
	s.material.shader = _shader
	s.material.set_shader_parameter("mask_tex", AssetLibrary._load_tex(dir + "/mask.png"))
	var c1: Array = s.meta.get("c1_base", [255, 255, 255])
	var c2: Array = s.meta.get("c2_base", [255, 255, 255])
	var base1 := Color(c1[0] / 255.0, c1[1] / 255.0, c1[2] / 255.0)
	var base2 := Color(c2[0] / 255.0, c2[1] / 255.0, c2[2] / 255.0)
	for pair in [["c1_base", base1], ["c1_tint", base1], ["c2_base", base2], ["c2_tint", base2]]:
		s.material.set_shader_parameter(pair[0], pair[1])
	return s


func ship(kind: String) -> SpriteSet:
	return _ships.get(kind)


func station(archetype: String) -> SpriteSet:
	return _stations.get(archetype)


func body(kind_id: String) -> Texture2D:
	return _bodies.get(kind_id)


func star_layer(layer_name: String) -> Texture2D:
	return _star_layers.get(layer_name)
```

## Reuse-or-create a child sprite pool keyed by id; anything not touched
## this frame is hidden (not freed - ship counts are small and stable).
func _pool_sprite(parent: Node2D, key: String, touched: Dictionary) -> Sprite2D:
	touched[key] = true
	var s: Sprite2D = parent.get_node_or_null(NodePath(key))
	if s == null:
		s = Sprite2D.new()
		s.name = key
		s.texture_filter = CanvasItem.TEXTURE_FILTER_NEAREST
		parent.add_child(s)
	s.visible = true
	return s


func _hide_untouched(parent: Node2D, touched: Dictionary) -> void:
	for child in parent.get_children():
		if not touched.has(child.name):
			child.visible = false
```

		while y < vp.y:
			var x := off.x
			while x < vp.x:
				draw_texture(tex, Vector2(x, y), Color(1, 1, 1, PARALLAX_ALPHA))
				x += tile.x
			y += tile.y
```

so stars drift opposite the flight direction correctly.
