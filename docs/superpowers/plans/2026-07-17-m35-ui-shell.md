# M3.5 PR 4 — UI Shell Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** The game speaks Rijay: three typography slots (pixel diegetic / reading sans /
stencil markings), a 90s-terminal amber-on-dark main menu + HUD skin, the D/H logo
revival, and app basics (window size/stretch, icon).

**Architecture:** Fonts are OFL files in `client/assets/fonts/` loaded at runtime
(`FontFile.load_dynamic_font`); a small `UiTheme` helper (static funcs on a new
`ui_theme.gd`) hands out fonts/colors/styleboxes so main.gd applies the Rijay dialect
to the existing three Labels without a scene rebuild. The main menu is a CanvasLayer
built in code (`main_menu.gd`), shown only when no `--username=` cmdline arg exists;
NetworkClient gains a `manual_login` gate + `request_login(u, p)`. The logo is generated
(`tools/artspike/logo.py`) like all other art.

**Tech Stack:** Python/resvg for the logo; Godot 4.7; OFL fonts: VT323 (pixel/terminal),
Inter variable (reading), Allerta Stencil (markings), fetched from google/fonts GitHub
with their OFL.txt licenses.

## Global Constraints

- Branch `m3.5-vibe-pass`, stacked. Game shell speaks Rijay and never reskins per run.
- Rijay terminal palette: amber `#ffb46b` on near-black `#120e08` panels, dim amber
  `#8a6a45` for secondary text, blue `#3b8de0` reserved for the logo/accents only.
- Typography slots: VT323 = small/diegetic (console readouts, HUD); Inter = large/
  reading (dialogue/menus later; menu subtitle now); Allerta Stencil = markings
  (menu title flavor, trade-panel header accent).
- Automation must not regress: `--username=` cmdline arg → menu never appears,
  auto-login exactly as today. Defaults-based auto-login ("pilot") is REMOVED only
  when the menu is up (no cmdline creds); harness flows all pass creds.
- Licenses ship next to the fonts (`client/assets/fonts/OFL-*.txt`).
- App basics: window 1280×720, stretch `canvas_items` + aspect `expand`, icon from the
  generated logo (`client/icon.png` — outside assets/ so Godot imports it normally).

### Task 3: Main menu + login gate

- [ ] network_client.gd: `var manual_login := false` set in `_ready` when no cmdline
  `--username=`; in the OPEN branch, only `_send_login()` when `not manual_login`;
  add:

```gdscript
## Called by the main menu: store credentials and log in now (or as soon as
## the socket opens). Safe to call again after an error_received retry.
var _menu_username := ""
var _menu_password := ""
func request_login(username: String, password: String) -> void:
	_menu_username = username
	_menu_password = password
	manual_login = false
	if _socket != null and _socket.get_ready_state() == WebSocketPeer.STATE_OPEN \
			and not _login_sent:
		_send_login()
```

  and `_login_credentials()` prefers `_menu_username` when non-empty.
